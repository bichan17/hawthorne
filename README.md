## getting started
'''cd''' to the '''src/''' directory and run '''grunt watch'''.
the site will get built in the '''build/deploy''' directory.




### what it does..
- compile my sass √
  grunt-contrib-compass

- watch for changes √
  grunt-contrib-watch

- copy over files √
  grunt-contrib-copy

- compile my JS √
  yui-compressor
